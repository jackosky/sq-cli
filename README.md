## Installation

### Prerequisites

- [Go](https://go.dev/doc/install) 1.21 or later
- A SonarQube Cloud account

### Build

Clone the repository and build the binary:

```bash
git clone https://github.com/jackosky/sq-cli.git
cd sq-cli
go mod tidy
go build -o sq main.go
```

## Configuration

Run the interactive setup to connect to your SonarQube Cloud account:

```bash
./sq auth login
```

This will open your browser to generate a user token and save your credentials to `~/.sq-cli.yaml`.

## Usage

### Projects

List all projects (first 50) in your organization:
```bash
./sq get projects
```

Filter projects by name or key:
```bash
./sq get projects --filter "my-service"
```

Create a new project:
```bash
./sq create project
```

### Issues

List issues for a specific project:
```bash
./sq get issues <project-key>
```
*Note: This checks the default branch by default. Use `--branch <name>` to specify a branch.*

### Pull Requests

List open Pull Requests and their Quality Gate status:
```bash
./sq get prs <project-key>
```
